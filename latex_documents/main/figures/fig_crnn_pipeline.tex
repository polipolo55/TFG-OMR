\caption[CRNN--CTC pipeline for monophonic OMR.]{%
  The CRNN--CTC pipeline applied to a single music staff, drawn in three
  phases. A \emph{visual encoder} (a ResNet18 convolutional backbone)
  turns the greyscale staff strip into a feature map and reshapes it into a
  left-to-right sequence of $W/4$ frames; a \emph{sequence model} (a
  bidirectional LSTM) contextualises each frame; and the \emph{CTC
  transcription} stage projects every frame to a distribution over the
  vocabulary (plus the blank symbol) and collapses the blank-padded
  alignment into a flat token sequence, without any explicit symbol
  segmentation. Solid arrows carry the data flow; the dotted arrow marks the
  fixed vocabulary that defines the output layer.
  Architecture introduced by \textcite{calvoZaragozaEndToEndNeural2018}.}
\label{fig:soa-crnn-pipeline}
