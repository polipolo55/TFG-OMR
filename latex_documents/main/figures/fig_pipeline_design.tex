\caption[Five-stage inference pipeline.]{%
  Horizontal data flow of the inference pipeline.  The
  \emph{visual front-end} (preprocessing $+$ staff detection)
  normalises the input page and emits, for every detected system, one
  staff-strip image and one chord-strip image.  Two parallel
  \emph{streams} (music and chord) feed each strip through an
  independent CRNN--CTC recogniser.  The \emph{output stage} repairs
  the music stream with a deterministic grammar validator and merges
  the two outputs into a JSON document.  Each block has a single
  responsibility and a typed input/output contract
  (\Cref{tab:stage-contracts}); the two streams share no parameters
  and are trained independently.}
\label{fig:pipeline-design}
